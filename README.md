SLF-DF is a reproducible pipeline for constructing a heuristically labelled dataset of Sybil-linked Front-running candidates in DeFi. It integrates Uniswap DEX transaction data from Dune with Blocknative mempool traces, correlates Sybil-related and non-Sybil transactions using temporal proximity, confirmation order, gas expenditure, and token-consistency constraints, and assigns \textit{victim}, \textit{front-runner}, and \textit{normal} labels.
